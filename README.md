# Nithish K Portfolio

## Overview

This is a personal portfolio website created by Nithish K to showcase skills, projects, certifications, and contact details. The portfolio is built with HTML, CSS, JavaScript, Bootstrap, and includes responsive navigation across multiple pages.

## Pages

- `index.html` - Home page with introduction and social links.
- `about.html` - About page describing experience, skills, and a downloadable CV.
- `skill.html` - Skills page highlighting technical competencies.
- `project.html` - Projects page presenting multiple web applications.
- `certifications.html` - Certifications page listing completed courses or credentials.
- `contact.html` - Contact page with methods to connect.

## Featured Projects

- Portfolio Website - Responsive portfolio built with HTML, CSS, JavaScript, and Bootstrap.
- Calculator - JavaScript,jquery calculator with a clean UI and basic arithmetic functions.
- E-commerce Landing Page - Modern landing page with responsive layout and interactive jQuery elements.
- Task Management App - React-based task organizer with add, edit, and delete features.
- React Login Page - Login interface built with React, form validation, and responsive styling.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- jQuery
- React
- Font Awesome

## How to Run

1. Clone or download the repository.
2. Open `index.html` in a web browser.
3. Navigate through the portfolio using the menu.
